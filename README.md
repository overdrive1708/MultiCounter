# MultiCounter

## 【簡単な説明】

5個のカウンタを同時に管理できるアプリケーションです。  
カウンタ選択：1,2,3,4,5,←,→  
カウントアップ：↑,+(テンキー)  
カウントダウン：↓,-(テンキー)  
カウントクリア：BackSpace,Delete,0  

## 【必要要件】

- Microsoft .NET Framework 4.7.2

## 【開発環境】

Microsoft Visual Studio Community 2019  
Version 16.10.4  

Microsoft .NET Framework  
Version 4.8.04084  

Prism.Unity  
v8.0.0.1909  
(NuGetを使用)
